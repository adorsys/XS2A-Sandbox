STEP 2: Payment cancellation authorisation

Payment cancellation authorisation for user with no SCA methods

4. Start payment cancellation authorisation with POST endpoint . Choose the same and you chose in STEP 1.

5. Enter payment id you have got from the previous response in STEP 1 in the field .

6. Fill the header fields with the following values:

Header Value
X-Request-ID 2f77a125-aa7a-45c0-b414-cea25a116035
PSU-ID YOUR_USER_LOGIN
PSU-IP-Address 1.1.1.1

Put in YOUR_USER_LOGIN the name of the user you chose (you can view users and their information in TPP-UI).

7. Execute the command and get 201 response and related information.
